Lesson 3 Pytorch Installation

1. Introduction to Pytorch

PyTorch is a specialized deep learning software library, using both GPU and
CPU. Automatic differentiation is achieved through a tape-based system at the
function and neural network layer level. This feature brings high flexibility and
speed as a deep learning framework and provides accelerated functionality
similar to NumPy. These redistributable components provided by NVIDIA are
Python pip Wheel installers for PyTorch, offering GPU acceleration and
support for cuDNN.
